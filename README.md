#Performance Analysis of Near-Field THz MU-MIMO Precoding with Power Focusing and Molecular Absorption

Near-field (NF) Terahertz (THz) communication with extremely large antenna arrays enables simultaneous distance- and angle-dependent beam focusing, offering an additional spatial multiplexing dimension beyond conventional far-field systems. This repository implements a comprehensive simulation framework for NF THz multi-user MIMO (MU-MIMO) systems by incorporating:

Near-field spherical wave propagation
Frequency-dependent molecular absorption based on the HITRAN model
Distance-angle beam focusing
Linear precoding techniques
Constant Modulus Constrained (CMC) precoding
Achievable rate and leakage power evaluation

The repository reproduces the numerical results presented in the accompanying paper.
